# Calculator

A basic calculator that performs arithmetic operations.

## Description
This project is part of TOP curriculum Foundation course final project and helps practice JavaScript logic and UI interaction.

## Built With
- HTML
- CSS
- JavaScript

## Features
- Addition, subtraction, multiplication and division
- Clear button

## What I Learned
- JavaScript functions
- Handling user input
- State management

## Known Issues

- Repeated press of equal (`=`) button do not repeat the last operation  
  (e.g. `3 + 5 =` shows `8`, but pressing `=` again should produce `13` then `18`)

## Future Improvements

- Add keyboard support
- Show visual feedback when operators are clicked
- Handle repeated equals (`=`) operations
- Improve decimal and large number handling
